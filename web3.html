What is Web3?

Web 3 technology involves the use of decentralized systems to store or process user activity on a website. The benefits of using this technology are that it allows user privacy and keeps their data out of the hands of big corporations. This technology can go hand in hand with Web 2.0 technology, but can also be its own separate category. The web versions that were made prior (web 1.0 and web 2.0) focussed on innovations to website creation and use, while web 3.0 focusses on decentralization and putting data on the blockchain, and making the data decentralized.

Implications of Web3

As of right now, the implications of Web 3 are indecisive since it is not yet mainstream. Just like most things, there are upsides and downsides. The upsides are more privacy and more control on the user end while the downsides include potential cybercrime and hate speech. At the end of the day, this technology is promising, but we’ll see if more corporations implement something that might be detrimental to their business.
